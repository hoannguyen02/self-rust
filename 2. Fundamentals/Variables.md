* Assign data to a temporary memory location
* Can be set to any value & type
* Immutable by default, but can be mutable by using mut keyword when declare
* Examples 
```
let my_age = 32;
let my_name = "Hoan";
let my_monthly_bill = 16.2;
let my_gender = M;

let mut my_happy = "Coding"
```
